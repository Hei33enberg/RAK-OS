# Hall of Fame — the open ledger of the Realm

Contribution to RAK OS is credited in the open. This is the ledger. Roles and level gates are defined in [`REALM.md`](./REALM.md).

## Levels

| Level | Realm role | Gate |
|---|---|---|
| L0 | Citizen | Show up — issue, MCP connection, local node. |
| L1 | Correspondent | 1 substantive merged PR. |
| L2 | Editor | 3+ substantive merged PRs (listed below). |
| L3 | Bureau Chief | 5+ PRs, sustained area ownership → merge rights. |
| L4 | Herald | Sustained propagation — Crown-nominated. |
| L5 | Node Sovereign | A live forked RAK node in production. |

## How to be added

Land substantive merged PRs. The bar for a Hall of Fame row (L2) is **3+ substantive merged PRs** — where "substantive" means a PR labelled `substantive` by a Bureau Chief or higher (this is the anti-farming rule; typo fixes are welcome but don't level you up). See [`CONTRIBUTING.md`](./CONTRIBUTING.md).
